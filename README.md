# CodeLens — AI Smart Code Reviewer

CodeLens is a browser-based AI tool that reviews code for **readability**, **structure**, and **maintainability** before it reaches human reviewers. A developer pastes a snippet, selects the language, and receives an instant analysis: three prioritized improvement suggestions with actionable fixes, one positive observation, and scored metrics across three dimensions.

Built as a single HTML file using the Claude API, it requires no backend or setup. The goal is to shift left on code quality — catching issues like SQL injection, poor naming, and callback hell automatically, reducing reviewer cognitive load and helping developers grow faster through consistent, structured feedback.
